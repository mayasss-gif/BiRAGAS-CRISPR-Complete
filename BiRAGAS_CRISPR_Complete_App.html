CRISPRi / CRISPRa Guide Designer Transcriptional Modulation

Targeting Windows

ModeWindow (rel. TSS)Mechanism
CRISPRi-50 to +300 bpdCas9-KRAB blocks transcription elongation
CRISPRa-400 to -50 bpdCas9-VP64/p65/Rta activates transcription
CRISPRi: dCas9 fused to KRAB repression domain. Guides target near TSS (+50 to +300 optimal). Blocks RNA Pol II elongation.

CRISPRa: dCas9 fused to VP64-p65-Rta (VPR) activation domain. Guides upstream of TSS (-400 to -50). Recruits transcriptional machinery.

RNA Base Editing Overview

EditorConversionDeaminaseWindow

A-to-I editing: Uses dCas13-ADAR2dd fusion. Creates A-C mismatch in guide RNA to direct editing. Inosine is read as guanosine.

C-to-U editing: Uses dCas13-APOBEC fusion. Directs cytidine deamination on RNA. No permanent genomic changes.

Installation & Setup Full Backend

1. Clone & Install

# Clone the repository git clone https://github.com/AyassBioscience/BiRAGAS-CRISPR-Complete.git cd BiRAGAS-CRISPR-Complete # Create virtual environment python3.12 -m venv .venv source .venv/bin/activate # Install dependencies pip install -r requirements.txt pip install -e .

2. Start the Server

# Start API server (enables full functionality) python3.12 -m Py_Codes_BiRAGAS_CRISPR --serve # Server runs at http://localhost:8000 # Open this HTML file in browser

3. DNA CLI Examples

# Design DNA guides python3.12 -m Py_Codes_BiRAGAS_CRISPR guide-design \ --gene BRAF --pam NGG --n-guides 4 # Full knockout strategy python3.12 -m Py_Codes_BiRAGAS_CRISPR knockout \ --gene TP53 --all-configs # Combination synergy python3.12 -m Py_Codes_BiRAGAS_CRISPR combo \ --gene-a BRAF --gene-b MEK1

4. RNA CLI Examples

# Design Cas13 RNA guides python3.12 -m Py_Codes_BiRAGAS_CRISPR rna-guide \ --gene KRAS --nuclease Cas13d --n-guides 4 # RNA knockdown strategy python3.12 -m Py_Codes_BiRAGAS_CRISPR rna-knockdown \ --gene MYC --nuclease CasRx # CRISPRi design python3.12 -m Py_Codes_BiRAGAS_CRISPR crispri \ --gene VEGFA --mode interference # RNA base editing python3.12 -m Py_Codes_BiRAGAS_CRISPR rna-base-edit \ --gene PCSK9 --editor ADAR2 \ --sequence AUGCGAUCGAACGAUCG
